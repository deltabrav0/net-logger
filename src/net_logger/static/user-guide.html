Net Logger User Guide

A quick guide for running nets, checking in stations, updating FCC data, and exporting records.

Opening Net Logger

Start the server, then open Net Logger in a browser. The default local address is:

http://127.0.0.1:8088

If the server was started on all LAN interfaces, another device on the same network can open the app by using the host computer's LAN address and configured port.

Basic use

  1. Enter a net name, frequency, and Net Control callsign.
  2. Press Start Net.
  3. Net Control is automatically added to Checked In.
  4. Check in stations with the single station lookup box, a card's Check in button, or drag-and-drop.
  5. Expand a checked-in card to edit traffic and notes.
  6. Press Stop Net when the net is finished.
  7. Use CSV export or Metrics as needed.

Starting a net

At the top of the page, enter the net name, frequency, and Net Control callsign, then press Start Net. If Net Control is not already known, Net Logger attempts a local FCC lookup when FCC data is configured. Net Control is then automatically added to the Checked In column for the active session.

Searching, adding, and checking in stations

Enter a callsign in the single station lookup box labeled Search or add station. Net Logger searches known stations first. If no known callsign matches, it then searches the local FCC database. If FCC data is found, the station is saved with FCC details; otherwise a callsign-only station is saved.

When a net is open, a station found or created from this box is checked into the active net automatically. When no net is open, the station list is filtered as you type and new stations are saved for later use.

You can also check in a known station by clicking Check in on the station card or dragging the station card from Known Stations to Checked In.

If a manually added station is later available in FCC records, click Update details on its Known Stations card. Net Logger looks up that station's callsign in the local FCC database and updates its name, location, grid, latitude, and longitude. Use Delete only when a station record is wrong or unwanted; deleting a station also removes saved check-ins for that station.

Editing traffic and notes

Expand a checked-in station card to edit whether the station has traffic, traffic details, and notes.

FCC database age and updates

The toolbar shows the local FCC database age, for example FCC database: 3 days old. If local FCC data is missing, the indicator shows that the database is unavailable.

Press Update FCC Database to download the current FCC amateur-license file, extract the local lookup files, and rebuild the callsign index. The update requires internet access and can take a few minutes.

Last Heard timestamps

Known station cards display Last Heard information when available. This helps Net Control quickly see which stations have checked in recently.

Stopping a net

Press Stop Net when the net is finished. The session and check-ins are saved in the SQLite database.

CSV export

Use CSV export to download saved net records for spreadsheet review, archival records, or external reporting.

Metrics

The Metrics view summarizes check-in activity by net and by week, month, or year.

Backing up data

Stop the server, then copy the SQLite database file. On macOS with default installed paths:

cp "$HOME/Library/Application Support/Net Logger/net_logger.sqlite3" ./net_logger_backup.sqlite3
